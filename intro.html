Introduction¶

PyPM provides a pypm.particlemesh.ParticleMesh object for solving forces with the particle mesh method.

ParticleMesh object is a state machine.

The typical routine for calculating force is

  1. Paint particles via paint()
  2. Real to Complex transform via r2c()
  3. Complex to Real transform via c2r(), applying transfer functions
  4. Read out force values via readout()
  5. go back to 3, for other force components

We provide a set of commonly used transfer functions in pypm.transfer.TransferFunction.
